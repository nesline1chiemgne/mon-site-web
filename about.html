Notre Mission

Ce projet démontre l'implémentation d'un pipeline CI/CD complet utilisant Jenkins, Vagrant et GitHub.

Technologies Utilisées

  • ✅ Vagrant - Virtualisation et provisionnement
  • ✅ VirtualBox - Environnement virtualisé
  • ✅ GitHub - Gestion de code source
  • ✅ Apache - Serveur web
  • ✅ HTML/CSS - Interface utilisateur

Fonctionnalités du Pipeline

2 Validation et tests
